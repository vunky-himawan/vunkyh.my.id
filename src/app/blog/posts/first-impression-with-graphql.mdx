## Nyobain GraphQL


Ini apa maksudnya? Ini schema pertamaku, aku bikin tipe sederhana:

```ts

  id: ID!
  title: String!
  completed: Boolean!
}

  todos: [Todo!]!
}
```

Pas nulis query ini, aku bingung banget, tapi begitu jalan, semua todo langsung keluar rapi! Terus ada resolver, aku mikir, 'Ini ngapain sih?'

```ts
@Resolver("Todo")

  constructor(private readonly todoService: TodoService) {}

  @Query(() => [Todo])
  async todos() {
    return this.todoService.getAllTodos();
  }
}
```

Awalnya ga ngerti ini buat apa, tapi pas nyambung ke service, ternyata cuma **gitu doang** wkwkwkâ€Š-â€Šga perlu rute macem-macem! Di frontend juga kaget, bodynya kok beda bangetâ€Š-â€Šbukan JSON biasa kayak REST, tapi harus nulis query dulu. Pusing deh awalnya.
